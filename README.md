# AI Resume Analyzer

AI Resume Analyzer is a web application that analyzes resumes using AI and provides feedback to improve them.

## Features
- Upload resume for analysis
- AI based skill and keyword detection
- Suggestions to improve resume
- User authentication (Login / Register)
- Stores previous resume analysis records
- Dashboard to view past analyses

## Tech Stack
- Python
- Flask
- HTML
- CSS
- JavaScript

## Purpose
This project helps students and job seekers analyze their resumes and improve them to increase chances of getting shortlisted for jobs.
